# 🛒 MiniShop JS

A small shopping cart system built with vanilla JavaScript using constructor functions and basic OOP concepts.

This project was built to practice object-oriented thinking and basic data structures in JavaScript.

Built as a learning project during my 365-day web development journey.

---

## 📦 What it does

- Creates products with name, price, and stock
- Adds products to a shopping cart
- Stores products inside an internal array
- Calculates the total price of the cart

---

## 🧠 Concepts practiced

- Constructor functions
- `this` keyword inside constructors
- Instance methods with `this.method = function()`
- `new` keyword to create instances
- Arrays inside constructors
- `push()` to add items
- `reduce()` to calculate totals

---

## 🚀 How to run it

1. Clone the repo: `git clone https://github.com/anudoranador87/minishop-js`
2. Open `index.js` in VS Code
3. Run with Node.js: `node index.js`
   — or open the browser console and paste the code directly
